# KCP documentation — build inputs

The human-readable KCP documentation lives at **<https://confluentinc.github.io/kcp/>**.

Everything under `docs/assets/` is the raw markdown input that `mkdocs build` consumes to produce that site. If you are here to read the docs, open the site above instead — it has proper navigation, search, and versioned releases.

If you are editing the docs, see `docs/assets/README.md` for the project overview, or `mkdocs.yml` at the repo root for the build configuration.
