## kcp docs

Show the documentation URL for this build

### Synopsis

Print the documentation site URL matching the running kcp binary's version.
Development builds resolve to the 'dev' alias; released builds resolve to their vX.Y.Z subdirectory.

```
kcp docs [flags]
```

### Examples

```
  kcp docs
```

### Options

```
  -h, --help   help for docs
```
