## kcp

A CLI tool for kafka cluster planning and migration

### Synopsis

A comprehensive CLI tool for planning and executing kafka cluster migrations to confluent cloud. Docs: https://confluentinc.github.io/kcp/dev/

### Options

```
  -h, --help      help for kcp
      --verbose   Enable verbose logging to console
```
