## kcp ui

Start the UI

### Synopsis

Starts the kcp UI.

```
kcp ui [flags]
```

### Examples

```
kcp ui --port 8080
```

### Options

```
  -h, --help                help for ui
  -p, --port string         Port to run the UI server on (default "5556")
      --state-file string   Path to a KCP state file to pre-load
```
